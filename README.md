# chrome_ext
# Excel Page Viewer - Chrome Extension (generated)

Включает:
- popup.html / popup.js — интерфейс экспорта
- content.js — извлечение контента на странице
- background.js — скачивание файлов
- readability.js — Mozilla Readability
- manifest.json — MV3 manifest

Как установить (локально):
1. Откройте `chrome://extensions/`
2. Включите режим разработчика (Developer mode)
3. Нажмите "Load unpacked" и выберите папку `excel_page_viewer_ext`
